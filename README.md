#SPACE URL

## install
server
- go to server directory use
```sh
cd server
```
- run command
```sh
node index.js
```
- server will running at [Breakdance](https://localhost:5000/)


client
- go to server directory use
```sh
cd client
```
- run command
```sh
npm start
```
- server will running at [Breakdance](https://localhost:3000/)
